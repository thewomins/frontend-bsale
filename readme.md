## Installation

```bash
yarn install
```

## How to use?

Open web in dev mode

```bash
yarn dev
```

### Scripts avaliable

Build web

```bash
yarn build
```

Preview

```bash
yarn preview
```

Format with prettier

```bash
yarn format
```

Check format with prettier

```bash
yarn check-format
```

Format with eslint

```bash
yarn lint
```
